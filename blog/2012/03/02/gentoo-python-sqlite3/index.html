Mar 2nd, 2012 | Comments

初用gentoo，总有一些新鲜的地方，以前在Ubuntu或Fedora中不用担心的事，到这里了都需要自己一手一脚的实现。昨天晚些时候，clone下以前的Django代码，计划开始做一些新的功能，但却遇到了sqlite3模块不存在的问题，于是开始安装sqlite，然后安装上了pyslqite，但仍没能解决问题。后来在这里找到了解决办法，在编译Python的时候，老是编译着3.1版的Python，幸而发现emerge的强大，可以对版本进行选择编译，总结下来，流程是这样的：
