html-upload 使用指南

独立 HTML 文档上传与管理

📦 简介

html-upload 是用于博客的独立 HTML 文档管理工具。上传的 HTML 文件保持原始样式，不被博客主题包裹。

访问地址：https://wwq0327.github.io/html/

🚀 触发方式

说出以下任一指令即可激活：

1
「上传 HTML」 上传新的 HTML 文档
2
「删除 HTML」 删除已有文档
3
「查找 HTML」 搜索已有文档
4
「/html-upload」 直接调用 skill

⬆️ 新增（上传）

上传新的 HTML 文档到博客。

文件名 → HTML 内容 → 描述 → 部署

文件名规则

  • 只能包含：a-z A-Z 0-9 - _
  • 空格和特殊字符会被自动过滤

重名处理

若文件名已存在，选择：

1
覆盖 替换原文件内容
2
重命名 换一个文件名
3
取消 放弃本次上传

🗑️ 删除

删除已有文档。文档按上传时间倒序显示。

显示列表 → 输入序号 → 确认 → 删除

示例列表

序号 文件名 描述 上传时间
1 my-document 我的文档 2026-05-10
2 claude-guide Claude 使用指南 2026-05-09
💡 删除前会确认

输入序号后需要输入 y 确认删除

🔍 查找

在文件名和描述中搜索关键词。

输入关键词 → 查看结果 → 选择操作

搜索后操作

  • 输入序号 — 查看文档详情
  • 输入 n — 继续搜索新关键词
  • 输入 q — 退出搜索

💬 对话示例

上传文档

你：上传 HTML
我：请输入文件名 → my-doc
我：请输入 HTML 内容 → [粘贴完整 HTML]
我：请输入描述（可选）→ 直接回车
我：上传完成！
    访问：https://wwq0327.github.io/html/my-doc.html

删除文档

你：删除 HTML
我：| 序号 | 文件名 | 描述 | 上传时间 |
    | 1 | my-doc | 我的文档 | 2026-05-10 |
你：1
我：确认删除 my-doc.html？（y/n）→ y
我：已删除

📝 注意事项

  • 上传的 HTML 文件保持独立，不显示博客导航栏
  • 文件名自动过滤非法字符，保留 a-zA-Z0-9-_
  • HTML 内容必须包含 <html> 或 <!DOCTYPE>
  • 描述字段可选，直接回车跳过
